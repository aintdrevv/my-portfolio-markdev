# Setup Notes

## Folder guide

- `frontend/` contains the website code
- `backend/` is where Xano files can live locally
- `docs/` stores notes so future changes are easier to manage

## Frontend

From the `frontend/` folder:

```bash
npm install
npm run dev
```

## Backend

When you are ready to connect Xano locally, this workspace already has a place for it in `backend/`.
